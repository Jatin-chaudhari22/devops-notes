##  File Management

## 1. Navigating the Filesystem

| Command | Description |
|---------|-------------|
| `pwd` | Print the current working directory. |
| `ls` | List files and directories in the current location. |
| `ls -l` | Detailed list with permissions, owner, and size. |
| `ls -a` | Show all files, including hidden files (starting with `.`). |
| `cd <directory>` | Change to a specified directory. |
| `cd ..` | Move up one directory level. |
| `cd ~` | Move to the home directory. |

---

## 2. File and Directory Operations

| Command | Description |
|---------|-------------|
| `mkdir <name>` | Create a new directory. |
| `rmdir <name>` | Remove an empty directory. |
| `rm -r <name>` | Remove a directory and its contents recursively. |
| `touch <file>` | Create an empty file or update file timestamp. |
| `cp <source> <destination>` | Copy a file or directory. |
| `mv <source> <destination>` | Move or rename a file or directory. |
| `rm <file>` | Delete a file. |
| `cat <file>` | View the content of a file. |
| `less <file>` | View file content page by page. |
| `head <file>` | View the first 10 lines of a file. |
| `tail <file>` | View the last 10 lines of a file. |

---

## 3. Searching Files and Content

| Command | Description |
|---------|-------------|
| `find /path -name "<filename>"` | Search for files by name. |
| `grep "pattern" <file>` | Search for a specific text pattern in a file. |
| `grep -r "pattern" /path` | Search recursively in all files under a directory. |
| `locate <filename>` | Quickly find files using the index database. |
| `updatedb` | Update the `locate` database (requires sudo). |

---

## 4. File Permissions and Ownership

Linux uses **read (r), write (w), and execute (x)** permissions for files and directories.

| Command | Description |
|---------|-------------|
| `ls -l` | Show file permissions, owner, and group. |
| `chmod 755 <file>` | Change file permissions. |
| `chown user:group <file>` | Change file owner and group. |
| `umask` | Show default permission mask for new files. |

**Tips:**
- Directories need **execute permission** (`x`) to access contents.  
- Use numeric (e.g., 755) or symbolic (e.g., u+rwx) modes with `chmod`.  

---

## 5. Disk Usage

| Command | Description |
|---------|-------------|
| `df -h` | Show disk usage per filesystem in human-readable format. |
| `du -sh <directory>` | Show total size of a directory. |
| `du -ah <directory>` | Show size of all files and subdirectories. |
| `ncdu <directory>` | Interactive disk usage analyzer (install if not present). |

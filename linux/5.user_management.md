## User Management

## 1. Viewing Users and Groups

| Command | Description |
|---------|-------------|
| `who` | Shows users currently logged in. |
| `w` | Displays logged-in users and their activity. |
| `id <username>` | Shows UID, GID, and group memberships of a user. |
| `groups <username>` | Displays all groups a user belongs to. |
| `cat /etc/passwd` | Lists all system users. |
| `cat /etc/group` | Lists all groups on the system. |

---

## 2. Adding and Removing Users

| Command | Description |
|---------|-------------|
| `sudo adduser <username>` | Adds a new user and creates a home directory. |
| `sudo useradd <username>` | Adds a new user (may require manual home dir creation). |
| `sudo passwd <username>` | Set or change the password for a user. |
| `sudo deluser <username>` | Removes a user (on Debian/Ubuntu). |
| `sudo userdel -r <username>` | Removes a user and their home directory. |

---

## 3. Managing Groups

| Command | Description |
|---------|-------------|
| `sudo addgroup <groupname>` | Creates a new group. |
| `sudo delgroup <groupname>` | Deletes a group. |
| `sudo usermod -aG <group> <user>` | Adds a user to a group. |
| `sudo gpasswd -d <user> <group>` | Removes a user from a group. |

---

## 4. User Permissions and Sudo Access

| Command | Description |
|---------|-------------|
| `sudo visudo` | Edit the sudoers file safely to grant sudo privileges. |
| `sudo usermod -aG sudo <username>` | Give a user sudo access (Debian/Ubuntu). |
| `sudo usermod -aG wheel <username>` | Give a user sudo access (RedHat/CentOS). |
| `chmod` / `chown` | Adjust file permissions and ownership for users and groups. |
